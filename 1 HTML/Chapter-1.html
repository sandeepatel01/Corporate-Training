Character Encoding

Character Encoding ka matlab hota hai ki web page ke text ko kaise samjha aur dikhaya jaye. Jab aap UTF-8 use karte ho, toh iska matlab hai ki aapka web page har tarah ke characters (jaise Hindi, English, emojis, special symbols) ko support karega.

Viewport

Viewport ka matlab hai device ki screen jisme aapka web page dikh raha hai, jaise mobile, tablet, ya desktop ki screen.

Agar aap viewport meta tag use nahi karte, toh web page desktop view me khulta hai, chahe user mobile pe hi dekh raha ho. Iska result ye hota hai ki page zoomed out lagta hai aur content chhota dikhai deta hai.

Viewport meta tag use karne se aap bata sakte ho browser ko ki web page ka size aur zoom level device ki screen ke hisaab se adjust ho.
