C++ Programming Terms

Object-Oriented Programming (OOP)
A programming paradigm supported by C++ that organizes code into objects with properties and behaviors.
Namespace
A feature in C++ used to group related code and avoid naming conflicts between identifiers.
STL (Standard Template Library)
A powerful library in C++ that provides generic classes and functions for data structures like vectors, stacks, and algorithms like sorting and searching.
Pointer
A variable in C++ that stores the memory address of another variable, enabling dynamic memory management.
Constructor
A special member function in C++ used to initialize objects when they are created.
