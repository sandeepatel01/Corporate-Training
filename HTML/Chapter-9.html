Common Video and Audio Formats

Format Type Description
MP4 Video A widely used format for video files, supporting high quality and compatibility across devices.
AVI Video An older format that supports high-quality video but larger file sizes compared to modern formats.
OGG Audio An open-source format offering good sound quality, commonly used in open web standards.
MP3 Audio A highly popular audio format known for its small file size and decent quality, widely supported.
WAV Audio A lossless format that provides high-quality audio but with larger file sizes, often used for professional recordings.
